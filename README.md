#  Repossitório para exercícios da matéria Pensamento computacional

##  Problema

Definir qual horário sair de casa para ir à faculdade, considerando trabalho em escala 3x3 e fatores externos.

### 1.Decomposição do problema

###  Rotina pessoal

*  Verificar se é dia de trabalho ou folga

*  Definir horário que precisa acordar

*  Tempo para se arrumar e sair
  

### Transporte

*  Verificar horário do ônibus

*  Tempo de deslocamento até o ponto

*  Tempo médio da viagem
  

### Fatores externos

*  Condições climáticas

*  Trânsito

*  Possíveis atrasos no transporte
*  Horário de início da aula


### 2. Identificação de padrões

*  Sempre existe um horário fixo de aula

*  Sempre existe um tempo de deslocamento

*  Sempre é necessário pegar um ônibus 

*  Sempre existe risco de atraso


###  3. Abstração

**Informações importantes:**

*  Horário da aula

*  Dia de trabalho ou folga

*  Horário do ônibus

*  Tempo de deslocamento

*  Condições externas

**Informações não essenciais:**

*  Roupa

*  Qual disciplina será ministrada
